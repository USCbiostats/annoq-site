About

The Annotation Query (AnnoQ) system is an integrated functional annotation platform for large-scale genetic variant annotation. The system is a large collection of over 700 million pre-annotated variants from the Trans-Omics for Precision Medicine (TOPMed) data Freeze 8 with sequence features by WGSA, functions by PANTHER, Gene Ontology, Reactome and gene enhancers from PEREGRINE. Currently only human variants are supported. The data can be accessed in one of the following ways.

  1. Browser.
  2. Swagger API data access using command line scripts.
  3. Software libraries AnnoQR available in the R programming language and annoq-py in the Python programming language.
